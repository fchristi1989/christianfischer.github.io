SimpleDigitalLocomotive
A private digital model railroad controlling software-project for MacOS-X


Description:
SimpleDigitalLocomotive is intended to control digital model railway locomotives and function models via the serial port (RS.422 (or RS.232 via USB)) or Ethernet port of an Apple Macintosh running MacOS 8.5 – 9.2.2 (SimpleDigitalLocomotive v0.2a4) and SimpleDigitalLocomotiveX (PPC and i386) running MacOS-X 10.3.9 / 10.4.11 / 10.5.2 / 10.6.1 / 10.7.0 – 10.11.0 or macOS Sierra and newer!

Supported Platforms:
MacOS-X
