SimpleDigitalLocomotive
A private digital model railroad controlling software-project for MacOS-X


Description:
SimpleDigitalLocomotive is intended to digitally control model railroad layouts via access to modern digital central units. This can be done via USB or Ethernet port of an Apple Macintosh running MacOS-X 10.5.3 or newer. The application includes the capability of several switchboards and unlimited memory roots and events. These roots can be defined freely beyond the features of comparable existing software.
