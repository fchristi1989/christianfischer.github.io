Wumpus Hunt
Mobile puzzle game made for Android devices


Description:
My first steps in programming for Android devices led to this mobile game. It is based on the old 1972 classic, but extended to further complexity. The programming was done with Android Studio. Key feature of the newest version is a completely self-developed online highscore system. Via php, highscores can be sent to a web server and viewed on every client.

Supported Platforms:
Android
